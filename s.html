Canadian Climate Data Accessibility (CCDA) Project

To start, search for a climate station ID, name of a place, or an approximate (+-0.25 degrees) decimal latitude/longitude of the area for which you want data/analysis.

The goal of this project is to allow easy access to Canadian, and eventually worldwide climate data and analytics. Please note that this project is still extremely young (verion 0.1), and that there will more features and data to come. As such there may be some data that is unaccessible, this means that it was unaccessible in the original database from which it came (ie. errors on the end of the original host's database). All data is unaltered.
